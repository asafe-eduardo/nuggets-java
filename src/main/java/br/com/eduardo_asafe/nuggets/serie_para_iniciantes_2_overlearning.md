## 2 - Overlearning

Sempre tem uma nova biblioteca, uma nova feature da linguagem ou uma nova peculiaridade no framework.

> Também tem os que sempre estão criando um novo projeto, um novo curso.

Você deve **dominar os fundamentos**, **dominar uma linguagem ou framework**, mas deve colocar esses conhecimentos em prática.

--- 

### A experiencia de mercado é Rei

Equilibre aprendizado com experiencia real.

- **freelancer**.
- colocando em prática alguma ideia (se você criar um site, provavelmente alguém vai clicar), 
- Candidate-se a vagas.
