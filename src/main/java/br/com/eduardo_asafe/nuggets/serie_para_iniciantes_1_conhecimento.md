## 1 - Falta do conhecimento do que realmente importa

Se você não sabe, você vai vim neste perfil, na postagem fixada, onde eu passo uma lista do 
que **eu entendo ser o básico de conhecimento** que um iniciante precisa para conseguir uma vaga de **Java Júnior**.

Mas, resumidamente, você precisa entender:

Os **princípios da linguagem**:
- Tipos
- Estruturas de repetição
- Fluxo condicionais
- Classes principais (ex: String, List)
- Orientação a objetos
- Saber **criar um projeto baseado em um framework**, se houver.
